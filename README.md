# 🛒 Talmass – iOS E-commerce App

Talmass is an elegant e-commerce application built with **UIKit**, following **MVVM + Coordinator** architecture. It provides users with a rich product catalog, detailed product views, a dynamic cart, profile editing, and smooth animations.

## 📱 Features

- 🛍 Product catalog with sorting and view modes (grid, list, large)
- 🔍 Product detail with add-to-cart and real-time counter
- 🧺 Dynamic cart actions: add, remove, increase, decrease, delete products
- 👤 Profile page with editable user info (name, phone, height, weight, etc.)
- 🎨 Custom TabBar with icons and titles
- 📡 REST API integration
- 🌐 Error handling and network check
- 🔐 Token storage with `UserDefaults`

## 🧱 Architecture

- MVVM (Model-View-ViewModel)
- Coordinator pattern for navigation
- Closures for UI updates and actions
- SnapKit for programmatic auto layout

## 🧰 Tech Stack

- `UIKit`
- `SnapKit`
- `GCD` (Grand Central Dispatch)
- `REST API` using `URLSession`
- `UserDefaults` for token/session
